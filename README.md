## 🧠 Features

* 🎯 Real-time face detection
* 📏 Distance estimation using face width
* 🧠 Smart labels:

  * 🔴 Too Close
  * 🟢 Perfect
  * 🔵 Too Far
* ⚡ Live FPS counter
* 🎥 Webcam-based interaction

```bash
python main.py
```

---

## ⚙️ How It Works

* Detects face using Haar Cascade
* Calculates face width in frame
* Estimates distance using focal length formula
* Displays status based on distance
